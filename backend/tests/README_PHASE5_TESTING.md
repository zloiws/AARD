# Запуск комплексных тестов Фазы 5

## Скрипт комплексного тестирования

`run_phase5_comprehensive_tests.py` - скрипт для запуска всех тестов по фазам и сбора статистики.

### Запуск

```bash
cd backend

# Запуск напрямую через Python (НЕ через pytest!)
python tests/run_phase5_comprehensive_tests.py
```

**Важно:** Скрипт нужно запускать напрямую через Python, а не через pytest, так как это не тестовый файл, а скрипт для запуска тестов.

### Что делает скрипт

1. Запускает все тесты по фазам:
   - Фаза 1-2: Базовые компоненты
   - Фаза 3: Расширенные сервисы
   - Фаза 4: Улучшение интеграции
   - Фаза 5: E2E тесты

2. Собирает статистику:
   - Количество пройденных тестов (PASSED)
   - Количество упавших тестов (FAILED)
   - Количество пропущенных тестов (SKIPPED)

3. Выводит итоговую сводку по каждой фазе и общую статистику

### Запуск отдельных тестов

Если нужно запустить отдельные тесты:

```bash
# E2E тесты Фазы 5
python -m pytest tests/integration/test_phase5_e2e_workflows.py -v

# Тесты Фазы 3
python -m pytest tests/integration/test_phase3_full_integration.py -v

# Тесты Фазы 4
python -m pytest tests/integration/test_phase4_integration.py -v
```

### Отладка

Если скрипт не показывает результаты правильно, проверьте:

1. Что тесты запускаются напрямую:
   ```bash
   python -m pytest tests/test_execution_context.py -v
   ```

2. Формат вывода pytest (должен содержать строку типа "X passed, Y failed, Z skipped")

3. Что все тестовые файлы существуют и доступны
