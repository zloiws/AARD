# Добавлена проверка загруженности модели перед запросом

## Реализовано

### 1. Метод проверки загруженности модели
- ✅ Добавлен метод `is_model_loaded()` в `OllamaClient`
- ✅ Использует Ollama API endpoint `/api/ps` для проверки загруженных моделей
- ✅ Проверяет наличие модели в GPU перед выполнением запроса
- ✅ Логирует предупреждение, если модель не загружена

### 2. Интеграция в процесс генерации
- ✅ Проверка выполняется перед каждым запросом к модели
- ✅ Логирование статуса загруженности модели
- ✅ Не блокирует выполнение, только предупреждает о возможной задержке

## Преимущества

1. **Избежание очередей**: Система знает, когда модель не загружена и может предупредить пользователя
2. **Мониторинг**: Логирование помогает отслеживать использование GPU
3. **Оптимизация**: В будущем можно реализовать предзагрузку популярных моделей
