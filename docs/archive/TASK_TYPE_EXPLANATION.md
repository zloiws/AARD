# Объяснение функции "Тип задачи"

## Что это такое?

**Тип задачи** (`task_type`) определяет, какая модель Ollama будет **автоматически выбрана** для обработки вашего запроса, если вы **не выбрали конкретную модель** вручную.

## Как это работает?

1. **Если модель НЕ выбрана вручную:**
   - Система использует тип задачи для выбора оптимальной модели
   - Например, для "Генерация кода" выберется модель, специализированная на коде (например, `qwen3-coder`)

2. **Если модель выбрана вручную:**
   - Тип задачи **игнорируется**
   - Используется выбранная вами модель независимо от типа задачи

## Доступные типы задач:

### 1. **Общий чат** (`general_chat`)
- Для обычных разговоров, вопросов, общения
- Используется модель общего назначения (например, `deepseek-r1`)
- **Когда использовать:** простые вопросы, беседа, общие запросы

### 2. **Генерация кода** (`code_generation`)
- Для написания, создания нового кода
- Используется специализированная модель для программирования (например, `qwen3-coder`)
- **Когда использовать:** "напиши функцию на Python", "создай компонент React"

### 3. **Анализ кода** (`code_analysis`)
- Для анализа, ревью, отладки существующего кода
- Используется модель для программирования (например, `qwen3-coder`)
- **Когда использовать:** "найди ошибку в этом коде", "оптимизируй этот алгоритм"

### 4. **Рассуждения** (`reasoning`)
- Для сложных логических задач, математики, размышлений
- Используется модель для рассуждений (например, `deepseek-r1`)
- **Когда использовать:** "объясни логику", "реши математическую задачу"

### 5. **Планирование** (`planning`)
- Для создания планов, структурирования задач
- Используется модель для рассуждений (например, `deepseek-r1`)
- **Когда использовать:** "составь план разработки", "распиши шаги проекта"

### 6. **Генерация текста** (`text_generation`)
- Для написания статей, документов, длинных текстов
- Используется модель общего назначения
- **Когда использовать:** "напиши статью", "создай документацию"

## Важно:

- Тип задачи работает только при **автоматическом выборе модели**
- Если вы выбрали модель вручную - используйте подходящий тип задачи просто для информации (или оставьте "Общий чат")
- Тип задачи помогает системе понять контекст и выбрать оптимальную модель
- В будущем это также может влиять на системные промпты и параметры генерации
