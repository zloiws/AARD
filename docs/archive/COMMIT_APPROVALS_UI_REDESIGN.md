# Переработка UI закладки "Утверждения"

## Реализовано

### 1. Статистика в header
- ✅ Общее количество утверждений
- ✅ Количество ожидающих утверждения
- ✅ Количество утвержденных
- ✅ Количество отклоненных
- ✅ Индикатор срочных (с таймаутом < 2 часов) с анимацией

### 2. Фильтры
- ✅ Фильтр по статусу (Ожидают, Утверждено, Отклонено, Все)
- ✅ Фильтр по типу (Планы, Артефакты, Промпты, Шаги выполнения)
- ✅ Сохранение выбранных фильтров в URL

### 3. Улучшенная информация
- ✅ Индикаторы срочности для утверждений с близким таймаутом
- ✅ Отображение статуса утверждения (утверждено/отклонено)
- ✅ Информация о том, кто утвердил
- ✅ Улучшенное отображение метаданных

### 4. Backend улучшения
- ✅ Метод `get_approval_statistics()` для получения статистики
- ✅ Метод `get_all_requests()` с поддержкой фильтров
- ✅ Обновлен endpoint `/approvals` для поддержки фильтров

## Измененные файлы

**Backend:**
- `backend/app/services/approval_service.py` - добавлены методы статистики и фильтрации
- `backend/app/api/routes/approvals_pages.py` - обновлен endpoint, добавлена передача статистики

**Frontend:**
- `frontend/templates/approvals/queue.html` - полностью переработан UI с статистикой, фильтрами и улучшенной информацией

## Результат

Теперь закладка "Утверждения" стала значительно более информативной:
- Видна полная статистика по утверждениям
- Можно фильтровать по статусу и типу
- Видны срочные утверждения
- Больше информации о каждом утверждении
