# Отладка формы

## Проблема
Форма не отправляется при нажатии "Отправить"

## Исправления

1. **Добавлен HTMX extension `json-enc`** в base.html
2. **Исправлен `handleSubmit`**:
   - Теперь возвращает `true` чтобы позволить HTMX продолжить отправку
   - Убрана блокирующая проверка сервера
   - Добавлена проверка на существование элементов

3. **Добавлено логирование** для отладки:
   - `htmx:configRequest` - показывает конфигурацию запроса
   - `htmx:beforeRequest` - перед отправкой
   - `htmx:afterRequest` - после отправки
   - `htmx:responseError` - ошибки
   - `htmx:sendError` - ошибки отправки

## Проверка

Откройте консоль браузера (F12) и попробуйте отправить сообщение. Должны появиться логи:
- `HTMX Request:` - показывает данные запроса
- `HTMX Before Request:` - подтверждает отправку
- `HTMX After Request:` - показывает ответ

## Возможные проблемы

1. Если не видно логов - HTMX не работает, проверьте подключение скрипта
2. Если видно `HTMX Request` но нет `Before Request` - форма блокируется
3. Если видно ошибки - проверьте ответ сервера
