### Контекст проекта: Визуализатор LLM-пайплайнов
**Цель:** Приложение для отладки и визуализации работы RAG/Agent-системы.
**Стек:** React  + TypeScript + Vite.
**Библиотеки:**
- Состояние: Zustand.
- Визуализация графов: `@xyflow/react` (React Flow).
- UI-компоненты: Shadcn/ui + Tailwind CSS.
- Иконки: Lucide React.
**Структура панелей:**
1.  Левая (50%): Чат с вкладками сессий.
2.  Центральная (40%): Визуализатор нод-графа.
3.  Правая (10%, фиксированная): Панель настроек модели (авто/ручной выбор).
**Главный принцип:** Все панели общаются через центральное хранилище Zustand. Бэкенд отправляет события (через WS/SSE), которые обновляют store, а UI реагирует.
